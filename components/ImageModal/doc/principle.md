## 图片预览框实现逻辑

### 宽图/高图判定原理

宽高比 > 780 / 500 ? 宽图

宽高比 < 780 / 500 ? 高图

### 普通预览

1.若只有src

    不不显示列表区域

2.若imgPool.length > 0

    显示列表区域，并对src进行indexOf操作从而确定currentImg的位置

3.若为宽图

    width 100%，

4.若为高图

    height 100%

### 放大操作
1.若为宽图

    height 100%

2.若为高图

    width 100%
